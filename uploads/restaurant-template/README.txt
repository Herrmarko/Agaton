RESTAURANT WEBSITE TEMPLATE
============================

A clean, responsive one-page HTML/CSS template inspired by the
structure of elegant fine-dining restaurant sites (hero, intro,
gallery, menu, ethos, team, events, contact, footer).

FILES
-----
- index.html        Complete page template with inline CSS
- README.txt        This file

HOW TO USE
----------
1. Open index.html in a browser to preview.
2. Replace placeholder text in square brackets [ ... ] with your
   own copy.
3. Replace image filenames (img/dish-1.jpg, img/team-1.jpg, etc.)
   with your own photography. Create an "img/" folder next to
   index.html.
4. Update the Google Maps iframe inside the .contact__map div.
5. Update sister venues and social links in the footer.

CUSTOMISING DESIGN TOKENS
-------------------------
Design tokens live at the top of the <style> block under :root{}:

  --color-primary       Accent colour (maroon #9C4044 by default)
  --color-black         Header background
  --color-white         Page background
  --color-text          Body text colour
  --font-display        Serif display font (DM Serif Display fallback;
                        swap for PP Editorial New via your own license)
  --font-body           Sans-serif body font (Sofia Sans Semi Condensed)

TYPOGRAPHY
----------
The template loads DM Serif Display + Sofia Sans Semi Condensed
from Google Fonts. If you have a licence for PP Editorial New,
update the --font-display variable and remove the Google Fonts
link for the serif.

SECTIONS
--------
1.  Sticky black header with logo, nav, Book Now button
2.  Fullscreen hero with large serif title and subtitle
3.  Two-button CTA row (Book / View Menu)
4.  Intro row with icon + title + paragraph
5.  5-column dish gallery
6.  Centred Menu section with button
7.  Two-column Ethos with oval-masked image
8.  Triple image row
9.  Panoramic image
10. Team grid (4 cards) with paginator dots
11. What's On — 4 event cards
12. Contact (maroon background, map + info)
13. Footer (logo mark, links, sister venues, acknowledgement)
